About this app

This app exists as a convenience tool for WHS students, parents, and teachers. After my family and I struggled with the WHS stationery list, I decided to create an app that would help others in my position.

This app is not affiliated with Wellington High School. It exists solely as a convenience tool. If you're interested in using this app for your school or club, please contact me, and I can help with setup.

Privacy

This app does not collect any data, "personal" or otherwise. The source code of this app can be found on GitHub for independant verification.
