About this app

This app exists a convenience tool for WHS students, parents and teachers. After my family and I struggled with the WHS stationery list, I decided to create an app that would help others in my position.

This app is not affiliated with Wellington High School. It exists a convenience tool. If you're interested in using this app for your school or club, please contact me.

If you are a developer and are interested in viewing the source code of this app, you can find the source code on GitHub.
